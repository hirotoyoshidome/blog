ゾンビプロセスとは

ふとした時にできてしまうあいつのことです。

sudo kill -9 PID

コマンドでも殺すことができず、サービスのリスタートとかを余儀なくされるあいつです。 そいつをPythonスクリプトで作成するやつを作ってみました。 ついでにDocker化もしてみました。

ps aux | grep defunct

で確認することができます。 監視スクリプト書くときの、テストように使ってください。
