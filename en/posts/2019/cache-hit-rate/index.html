Cache Hit Rate Measurement Tool

2019-09-15
  • #engineering
⇄ 日本語
  • 2019/09/15 When designing web infrastructure you often serve static content via a CDN (Sakura CDN, AWS CloudFront, etc.). I built a tool that measures the cache hit ratio per page (URL) and highlights uncached content.

https://github.com/hirotoyoshidome/hit-rate

It is containerized, but because it uses Selenium, certain chromedriver versions may not work. Dependencies are just selenium and requests.
